Download (scan)

Pull the catalogue from Syncee into Postgres. Runs headless; needs a valid auth session.

Enrich

Fetch real shipping/description/stock. Skips already-enriched — repeat until 0 left.

Score

Run after enriching. Suppliers first (gates + score), then products. The pricing overrides re-price + re-score instantly — no rebuild needed.

rrp: sell at Syncee's suggested price (reject under min margin). · target_margin (uses Target margin %): price = cost ÷ (1 − fees − margin). · markup (uses Markup ×): price = landed cost (supplier+shipping+fees) × multiple → margin% = 1 − 1/×, e.g. 1.6 → 37.5%, 2.0 → 50%, 2.5 → 60%. Each field only applies to its mode; the RRP delta on the product card shows how far the price lands from market.

Select assortment

Build a curated candidate batch (diversity + per-supplier cap) from scored, reviewed products.

Publish

Prep the assortment (SEO + generative images), then push to Shopify. Push is a dry-run unless you tick Apply.

Export CSV

Download the current data straight from Postgres.
